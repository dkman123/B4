This folder holds the configuration files needed to build a B4 win64 installer

REQUIREMENTS:
    Environment :
        * a Windows OS
        * with python2.6+
        * with pymysql
    Tools:
        * cx_Freeze
        * InnoSetup
        * InnoSetup preprocessor
        * ISTools (optional)
